# TCP/IP

TCP: é um protocolo central da internet que garante a entrega confiável, ordenada e sem erros de dados entre dispositivos. Ele funciona na camada de transporte, quebrando mensagens em pacotes (segmentos), gerenciando o fluxo de informações e reorganizando os dados no destino.

IP é o que define o endereço e o caminho do pacote de dados, como "192.168.0.1", atribuído a cada dispositivo (computador, celular, impressora) conectado a uma rede, seja ela local ou a internet

Endereço IP: 123.89.46.72 32 bits= 4 bytes

ipv4: principais protocolos dos métodos de interligação de redes baseados em padrões na internet

possuí 5 classes (de A á E)

ipv6: é um rotulo numérico que identifica e localiza a interface de rede de um computador (oito grupos de quatro dígitos) 340 undecilhões de endereços 75 trilhões a mais do que o ipv4

128 bits = 16 bytes.
